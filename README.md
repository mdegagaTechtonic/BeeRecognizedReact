# BeeRecognized
### An Employee Recognition App to Enhance Workplace Culture
### TeamBee's Capstone project


## BeeRecognized Styling Convention:

### Acronyms

```
L: login
UP: user profile page
RS: recognition sent page
RR: recognition received page
```
